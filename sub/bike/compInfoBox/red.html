The rear derailleur in SRAM’s 12-speed RED eTap AXS 2X Electronic Post Mount HRD Groupset uses a clever fluid damping system, so that the one derailleur works for all possible setups, be those road, gravel, one front ring or two. Thus the drivetrain can be set up exactly as required for the ride without the need for a different rear derailleur each time. As part of the Orbit chain management system it does away with the need for a clutch mechanism and simply delivers fast, efficient and quiet changes whenever called upon.

The front derailleur has been redesigned also, and whilst it still utilises SRAM’s unique Yaw mechanism it now allows greater clearance for larger rear tyres. The battery is still interchangeable with the battery for the rear derailleur should the need ever arise, although with a range of 1,000km this shouldn’t be a frequent requirement.

The AXS app allows the customisation of certain functions within the drivetrain and gear changes as well as providing information about the battery’s health. Options such as Sequential or Compensating shift modes can be accessed depending on personal preference.
